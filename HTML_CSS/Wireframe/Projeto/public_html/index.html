Sobre nós

A escola Santa Mônica tem como objetivo sempre entregar aos alunos o melhor aprendizado que possamos oferecer, trabalhamos com o desenvolvimento cognitivo, afetivo, emocional, etc. Nossa equipe está sempre disposta a promover brincadeiras e interações entre os alunos para que o ambiente se torne o mais amigável e divertido possível, deixando os alunos serem livres para pensar, aprender e até mesmo criar. Oferecemos também espaços recreativos, onde os alunos aprendem um pouco mais sobre outras culturas, prática de esportes, atividades de lazer, entre outros.
